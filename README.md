# Resumo:
# Uma plataforma de geolocalização e auxílio ao descarte de resíduos especiais (logística reversa). O projeto consiste em um mapa interativo que conecta o cidadão aos pontos de coleta corretos para itens que não devem ir para o lixo comum, como óleo de cozinha, pilhas, baterias, eletrônicos e pneus.
